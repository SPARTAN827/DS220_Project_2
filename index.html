Video Game Reviews Analysis

Made by Jesse Zhang for Game Reviews Analysis

Project Overview

This project analyzes a comprehensive dataset of video game reviews to uncover insights about the gaming industry, player preferences, and factors that influence game ratings. Using Python and various data science libraries, we explore patterns and trends in video game success metrics.

Key Features

  • Analysis of 47,774 video game reviews
  • Exploration of pricing trends and their impact on ratings
  • Investigation of quality metrics influence on game success
  • Comparison of multiplayer vs single-player games
  • Developer performance analysis

Research Questions

  • What is the relationship between a game's price and its user rating?
  • How have video game ratings changed over time (2010-2023)?
  • Which genres consistently receive the highest user ratings?
  • Do multiplayer games receive higher ratings than single-player games?
  • What combination of quality factors has the strongest impact on ratings?
  • Who are the most consistent developers in maintaining high user ratings?
  • Is there a correlation between game length and user ratings?
  • How do ratings vary across different age group targets?
